# Fantasy-Combat
A terminal-based fighting game with lineups of various fantasy characters that battle until only one remains.

# Quick Start
To use locally, clone or download this repository to your local machine and branch. 

Instructions for cloning: [Cloning Help](https://help.github.com/articles/cloning-a-repository/)

run: **make**

run: **./fantasy**

To view repo: [Goto GitHub](https://github.com/JacobLCarter/Fantasy-Combat)
